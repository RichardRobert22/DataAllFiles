---------------- crear router wifi pc ----------------------------

netsh wlan set hostednetwork mode=allow ssid=name_ssid key=password

---------------- Activar red wifi  --------------------------------

 
netsh wlan start hostednetwork

--------------- Desactivar red wifi ------------------------------

netsh wlan stop hostednetwork
